# Task Manager App

Build using - ng new task-manager
version - @latest - v12.2.0

# Run App

<!-- runs on PORT:4200 -->

npm start or ng serve

# Run the server

<!-- runs on PORT:5000 -->

npm run server

# Functionality

1. User can add task
2. User can toggle reminder by Double Clicking on task
3. User can navigate between About page and Home page
